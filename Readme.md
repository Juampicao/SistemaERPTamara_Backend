Sistema ERP para una Pyme Argentina. 




Instalaciones: 
1) npm init
2) npm i express
3) npm i -D nodemon
4) npm install mongoose
5) npm i dotenv (variables entorno)
6) npm i bcrypt (hasheo)
7) npm i jsonwebtoken (Token)
8) npm i cors (localhost 3000 y 4000)
9) Git ignore .env
10) npm install nodemailer
